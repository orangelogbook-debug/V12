Adding a Serial or Machine ID

Serial or Machine IDs can be added manually for new machines, or whenever you cannot find the machine in the app. If a number already exists, the app will prevent you from creating a duplicate.

Character Requirements

The ID must contain 7 to 12 alphanumeric characters. The app already blocks anything outside this range automatically.

ATM Machine IDs

For ATMs, use the machine ID provided by the customer. Examples:

  • BDCQCxxx
  • AFxxxxxx

CRU Serial Numbers

For CRUs, use the official 7‑ or 12‑character serial number. Examples:

  • Old CM18: 000xxxx
  • New CM18: CT18xxxxxxxx

Location Details

For NRT contracts or large sites with multiple technicians, use the customer’s machine ID and your own notes to track the exact location. Example:

  • GCP01 — second floor, right of the stairs

This information is entered in the Location details field when creating a service request. It is saved permanently and will always appear in the machine’s history until you update it.
